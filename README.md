# Console-Chess-Game
Simple Console Chess Game

That's a simple console chess game which I developed when I was a student using C++.


This project is a good approach of the actual chess game,some bugs might appear in decent use cases.


Warning : All the console messages are displayed with greek characters.

Instructions for use :

First Step: You need to Compile&Run the main.cpp file

Second Step: You need to first select the pawn by giving the X and Y coordinates

Third Step: You need to give the desired position you want to move the pawn by giving the exact X and Y coordinates of this position
